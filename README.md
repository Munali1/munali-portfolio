# Munali Shakya Portfolio

Modern, responsive portfolio site for a backend, cloud, and AI-focused software engineer. Built with Next.js (App Router), Tailwind CSS, and Framer Motion.

## Features
- Professional, developer-centric design
- Responsive layout optimized for recruiters and hiring managers
- Highlight sections for backend, cloud, and AI experience
- Dark/light mode toggle
- Modular, easy-to-edit content

## Tech Stack
- Next.js
- React
- Tailwind CSS
- Framer Motion
- TypeScript

## Getting Started
Install dependencies:

```bash
npm install
```

Run the dev server:

```bash
npm run dev
```

Open `http://localhost:3000` in your browser.

## Customize Content
Update content in:
- `src/app/page.tsx`

## Deploy
Deploy easily on Vercel or Netlify.

## Notes
Add your CV file:
- `public/your-CV.pdf`

Configure email forwarding for the contact form:
- `SMTP_HOST`
- `SMTP_PORT`
- `SMTP_USER`
- `SMTP_PASS`
- `CONTACT_FROM`
- `CONTACT_TO` (set to `your email`)
